HTML

  • The head element contains information, or metadata, that helps the browser to render the page correctly.
  • The body element contains the content that is shown to the user.
  • !DOCTYPE is not case-sensitive because it is a declaration. Declarations are not case-sensitive; HTML elements are case-sensitive.
  • The title element defines the title for a webpage. This element is required and is extremely important for Search Engine Optimization (SEO).
  • Search engine optimization (SEO) is the process of improving the ranking of a website in search engines such as Google or Bing.
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.

CSS

  • Css: Cascading Style Sheets
  • You can do inline styles
  • You can link an external stylesheet in your head element
  • color: color or your text
  • font-size: changes the size of your font
  • text-align: changes your text alignment
  • border: adds a border around an element
  • margin: space around an element
  • padding: space between the internal content and the edge of an element
  • width: how wide an element it
  • box-shadow: CSS property adds shadow effects around an element's frame

Git

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • Git commands always start with git and are used to save and track versions of your code.
  • Use git clone [paste copied SSH value]: to create a clone of a repository
  • git status: check which branch you are currently working on
  • checkout: command is to move the working branch to a new branch.
  • -b: flag creates a new branch.
  • git checkout -b branchname: put the commands together to create a new branch
  • git add -A: . The add command adds modifications in the current working branch to the staging area. The -A flag indicates that we want to add all changes. The staging area informs Git that the changes will be added in the next commit.
  • git commit -m "....": describes what the commit contains

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Math.floor() and Math.random() are both properties of JavaScript's built-in Math object that can be used together to generate a random number.
  • It is common practice to indent the for loop code below the function name, within the curly brackets, to add visual structure to your code
  • Array: A way to store a group of related data
